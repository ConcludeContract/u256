# U256 

Pure Move language implementation of U256 numbers.

Would be nice to help with the `TODO` list, you can see it in the [header comment](sources/U256.move).

* Features:
  * mul
  * div
  * add
  * sub
  * shift left
  * shift right
  * compare
  * if math overflows the contract crashes.

### Build

    aptos move build

### Test

    aptos move test
